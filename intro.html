MÓDULO 1

Instalación y reconocimiento del programa R y R-Studio

Lección 1

Duración del video: 9:59

Actividades a realizar

En este primer video conoceremos como instalar y configurar el programa R y su interfaz de R-Studio, dejandolo listo para empezar a trabajar.

La actividad a realizar es de tipo formativa, por tanto, deben realizar la instalación de los programas según lo visto en el video para así poder avanzar en las siguientes lecciones.

Primero instalar el programa R desde este link: https://cran.dcc.uchile.cl/ para Windows o para MAC usar este otro link: https://cran.r-project.org/bin/macosx/

Posteriormente instalar la interfaz de edición de código integrada R-Studio en este link: https://posit.co/download/rstudio-desktop/

Para una revisión más detallada sobre el tema de instalación puede consultar el capítulo 2 del libro: R para principiantes de Juan Bosco Mendoza Vega en el siguiente link: https://bookdown.org/jboscomendoza/r-principiantes4/

Lección 2

Actividades a realizar

Realizar la configuración del ambiente de trabajo en el R-Studio según lo explicado en el video, y cree un proyecto de R con un Script de R, listo para trabajar.
